# DEBA.StockApp (MVP)

Ce dossier contient le MVP minimal de l'application DEBA.StockApp.

Étape 1: DatabaseService + schema SQL ont été ajoutés.

Pour initialiser la base depuis l'application :

1. Installer les packages NuGet.
2. Créer une instance de `DEBA.StockApp.Data.DatabaseService` et appeler `InitializeAsync()`.
